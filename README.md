# Jetpack-Joyride
(En)
Jetpack Joyride Clone (Raylib)
This project is a 2D endless runner game inspired by Jetpack Joyride.  
It is developed in C++ using the Raylib graphics library.

Gameplay:
The player uses a jetpack to fly.
Coins can be collected to increase the score.
Laser zappers must be avoided.
The objective is to survive as long as possible and reach the highest score.

 Controls:
SPACE or Mouse Click→ Fly
P → Pause


(Tr)
Jetpack Joyride Klonu (Raylib)
Bu proje, Jetpack Joyride'dan esinlenilmiş 2 boyutlu sonsuz koşu oyunudur.

Raylib grafik kütüphanesi kullanılarak C++ dilinde geliştirilmiştir.

Oynanış:
Oyuncu uçmak için bir jetpack kullanır.
Puanı artırmak için para toplanabilir.
Lazer tuzaklarından kaçınılmalıdır.
Amaç, mümkün olduğunca uzun süre hayatta kalmak ve en yüksek puana ulaşmaktır.

Kontroller:
BOŞLUK veya Fare Tıklaması → Uçma
P → Duraklat
